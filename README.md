# Lexova

An interactive music discovery platform designed to streamline how users explore and stream independent artists.

This is a code bundle for Lexova Prototype. The original project is available at https://www.figma.com/design/Cvo7VsSjY41Zfp2N0Zw5jA/Lexova-Prototype.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
